# Password Generator App

This is a simple password generator built using Python and Tkinter for the GUI. The app allows users to generate a random password based on their selected criteria (length, uppercase, lowercase, digits, symbols) and copy it to the clipboard.

## Features
- Select password length
- Include uppercase, lowercase, digits, and symbols
- Copy generated password to clipboard
- Simple, easy-to-use GUI built with Tkinter

## Requirements

Install the required libraries by running the following command:

git bash
pip install pyperclip
